## üöÄ Getting Started

### Prerequisites

- Modern web browser
- Git (for cloning)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Rishabh-Raj-Pathak/portfolio.git
```

2. Navigate to the project directory:

```bash
cd portfolio
```

3. Open in your preferred browser:

```bash
# Open index.html directly or use a local server
# For VS Code users:
code . && live-server
```

## üìù Customization

This portfolio is built with modularity in mind. To customize:

1. **Personal Info**: Update content in `index.html`
2. **Styling**: Modify variables in `assets/css/style.css`
3. **Projects**: Add new projects in the portfolio section
4. **Assets**: Replace images in `assets/images2/`
